# 🌎 LinguaSphere

Um site moderno para uma escola de idiomas fictícia, desenvolvido com HTML, CSS e JavaScript.

---

## 📖 Sobre o Projeto

O LinguaSphere foi desenvolvido com o objetivo de simular uma plataforma profissional de ensino de idiomas. O site apresenta informações sobre cursos, testes de nível, matrículas e canais de contato, oferecendo uma experiência intuitiva e responsiva para os usuários.

---

## ✨ Funcionalidades

* Página Inicial interativa
* Catálogo de cursos
* Teste de nível de idiomas
* Formulário de matrícula
* Formulário de contato com validação em JavaScript
* FAQ interativo
* Layout responsivo para dispositivos móveis
* Design moderno com identidade visual própria

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Fonts (Poppins)

## 🚀 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/LinguaSphere.git
```

2. Abra a pasta do projeto.

3. Execute o arquivo:

```text
Index.html
```

em seu navegador.
